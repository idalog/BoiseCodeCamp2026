## About

This repository contains the slides for a talk presented at **Boise Code Camp 2026** (May 2, 2026, College of Western Idaho). 

The session explores how to build, run, and customize powerful **AI agents** entirely on your local machine — no cloud dependencies, no subscriptions, and full data privacy.

### Topics Covered (based on title & context)
- Local LLM setup (Ollama, etc.)
- Building autonomous agents
- Tool integration and workflows
- Running agents offline with privacy and control
- Practical demos and architecture patterns
- Tips for performance on consumer hardware
